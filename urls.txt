User-facing:

/ (HTML)
  * New Game button

/status (HTML)
  * start page for chaser
  * start page for chasee
  * target page

API:

/chasee/current returns current page or "chaser"/"chasee" if the game is over
/chasee/target returns target page   or "chaser"/"chasee" if the game is over
/chasee/update/http://... sets current page

/chaser/update/http://... sets current page
